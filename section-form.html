Section 4.2
Error Fields

When validation fails for an input, wrap the label and input in a div.is-error and include a div.error-msg below the input.

Wrong!! :)

HTML

<div class="is-error">
  <label for="error-field">Name</label>
  <input type="text" id="error-field" name="error-field" />
  <div class="error-msg">Wrong!! :)</div>
</div>

Section 4.3
Fieldset and legend

Below are form fields!!

HTML

<fieldset>
  <legend>Below are form fields!!</legend>
  <label for="field">Label</label>
  <input type="text" id="field" name="field" />
</fieldset>

Section 4.4
Input Hints

Use input hints to offer hints or instructions to help users complete a form field.

Full, legal name

HTML

<label for="input-hint">Name</label>
<input type="text" id="input-hint" name="input-hint" />
<span class="input-hint">Full, legal name</span>
