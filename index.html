Style Guide

This style guide was automatically generated from KSS-style comments in the CSS/SASS files using kss-node.

  • The styles and components in this web project starter kit are from Springboard.
  • Edit this overview page by editing sass/styleguide.md. All other pages are automatically generated from KSS comments in the SASS source files.
  • If you want to change the look of the style guide, you can edit the template found in the styleguide-tmpl folder.
  • To regenerate the styleguide run the bash script styleguide.sh (or copy and paste the kss-node command inside the script into your terminal)
