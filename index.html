SVG Icons (svg sprite)

To create icon sprite:

  1. Export from Sketch
  2. Run svgo -f /folder-of-svgs to clean/compact svg markup
  3. Create SVG sprite on icomoon.io

Sprite must be inline (in HTML doc head) to support IE8-11. Or use SVG for Everybody or svgxuse for external SVG sprite.
