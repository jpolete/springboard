Section 5
Media

Section 5.1
Image

Images are set to max-width: 100% so that they’ll flex with the width of their container.

HTML

<img src="https://upload.wikimedia.org/wikipedia/commons/5/54/De_Alice%27s_Abenteuer_im_Wunderland_Carroll_pic_23_edited_1_of_2.png" alt="The Cheshire Cat">

Section 5.2
Figure

Use a figure anytime you have "...an image, an illustration, a diagram, a code snippet, or a schema that is referenced in the main text, but that can be moved to another page or to an appendix without affecting the main flow" https://developer.mozilla.org/en-US/docs/Web/HTML/Element/figure

HTML

<figure>
  <img src="http://upload.wikimedia.org/wikipedia/commons/thumb/d/da/Alice_par_John_Tenniel_02.png/392px-Alice_par_John_Tenniel_02.png" alt="" />
  <figcaption>The White Rabbit</figcaption>
</figure>

Section 5.3
Instrinsic Media Size

Defines image or iframe-embedded video size even before the media loads. Minimizes page repaints, reflows and maintains aspect ratio on screen resize.

http://daverupert.com/2015/12/intrinsic-placeholders-with-picture/

HTML

<picture class="intrinsic intrinsic--square">
  <source media="(min-width: 500px)" srcset="http://placehold.it/1000x1000?text=Square">
  <img class="intrinsic-item" srcset="http://placehold.it/300x300?text=Square" alt="Square Image">
</picture>

<hr>

<div class="intrinsic intrinsic--4x3">
  <img class="intrinsic-item" srcset="http://placehold.it/1200x900?text=4x3" alt="4x3 Image">
</div>

<hr>

<div class="intrinsic intrinsic--16x9">
  <iframe width="560" height="315" src="https://www.youtube.com/embed/XnBbjc5hmho" frameborder="0" allowfullscreen></iframe>
</div>

Section 5.4
Native Video

Native HTML video element. Videos are set to max-width: 100% so that they’ll flex with the width of their container.

HTML

<video autoplay controls src="public/video.mp4">
  Your browser does not support the <code>video</code> element.
</video>
