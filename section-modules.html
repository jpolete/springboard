Section 6.4
Message Box

Use to alert the user about a current or potential outcome such as success, failure or a warning. There are four types of messages.

Message Box

I am an important message! Links are styled to match the look of the warning box.

More important than your standard message. Warns of potential danger.
.msg--warning

Message Box

I am an important message! Links are styled to match the look of the warning box.

An error has occurred
.msg--error

Message Box

I am an important message! Links are styled to match the look of the warning box.

A success message
.msg--success

Message Box

I am an important message! Links are styled to match the look of the warning box.

HTML

<div class="msg [modifier class]">
  <h2 class="notice__heading">Message Box</h2>
  <p>I am an important message! 
  Links are <a href="https://developer.mozilla.org/en-US/docs/Web/CSS">styled to match</a> 
  the look of the warning box.</p>
</div>

Section 6.5
Old Browser Notice

A notice banner for older browsers encouraging users to upgrade. Really only targets old IE right now. Message is set to display: none unless it's a child of an element with a class of ie6, ie7, ie8, ie9. Use IE conditional comments to add these classes to the html root element.

You’re using a pretty old web browser. This site (and probably others) might not always look quite right. Why not grab the latest version of your web browser here?

HTML

<div class="ie8">
  <div class="old-browser-notice">
    <p>You’re using a pretty old web browser. This site (and probably others) might not always look quite right. Why not <a href="http://browsehappy.com/">grab the latest version of your web browser here</a>?</p>
  </div>
</div>

Section 6.6
Social Media Icons

Icons for common social media sites. Sprite must be inline (in HTML doc head) to support IE8-11. To use an external sprite with older IE versions you’ll need “SVG for Everybody” or “svgxuse”.

svgxuse - https://github.com/Keyamoon/svgxuse

SVG for Everybody - https://github.com/jonathantneal/svg4everybody

Use role, aria-label, and title for accessibility. See source code and this Stack Overflow discussion. http://stackoverflow.com/questions/4697100/accessibility-recommended-alt-text-convention-for-svg-and-mathml
