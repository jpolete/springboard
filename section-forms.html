Section 3.1
Form layout

Each line of a form should be wrapped in a <div.form-line>. A single label/input on a line should be wrapped in a <div.form-input-full>. For two label/inputs on a line wrap each in <div.form-input-half>. For three inputs wrap each in a <div.form-input-third> and for four inputs each should be wrapped in a <div.form-input-quarter>.

Example

HTML

<div class="form-line">
  <div class="form-input-half">
    <label for="FirstName">First Name</label>
    <input type="text" id="FirstName" name="FirstName" value="" />
  </div>
  <div class="form-input-half">
    <label for="LastName">Last Name</label>
    <input type="text" id="LastName" name="LastName" value="" />
  </div>
</div>
